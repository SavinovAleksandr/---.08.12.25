РАЗМЕЩЕНИЕ НА ХОСТИНГЕ (reg.ru):
---------------------------------
1. Распакуйте архив на компьютере
2. Загрузите ВСЕ файлы и папки в корневую директорию сайта на reg.ru
3. Сохраните структуру папок (css/, js/, img/)
4. Убедитесь, что файл index.html находится в корне сайта
5. Откройте домен в браузере

ВАЖНО:
------
- PHP скрипт send-email.php будет работать только на хостинге с поддержкой PHP
- На GitHub Pages форма автоматически переключится на mailto:
- На reg.ru форма будет отправлять письма напрямую на info@digital-tribe.ru
