Проект: Личный сайт-визитка Олега Соловьёва

Структура:
- index.html — главная страница
- privacy.html — политика конфиденциальности
- css/styles.css — стили
- js/script.js — логика (меню, модальное окно, отправка формы через mailto)
- img/hero.jpg — фото Олега
- img/logo-dt.png — логотип Digital-Tribe

Деплой:
1. Загрузите все файлы и папки из архива в корневой каталог сайта на хостинге (например, reg.ru).
2. Убедитесь, что стартовый файл называется index.html.
3. Откройте домен в браузере.
