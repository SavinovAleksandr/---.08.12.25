ИНСТРУКЦИЯ ПО РАЗМЕЩЕНИЮ САЙТА НА REG.RU
==========================================

⚠️ ВАЖНО: Это НЕ WordPress сайт! Это статический HTML сайт.

СТРУКТУРА ФАЙЛОВ:
-----------------
Все файлы должны быть в папке /www (корневая папка домена)

/www/
├── index.html          ← Главная страница (ОБЯЗАТЕЛЬНО в корне!)
├── privacy.html
├── robots.txt
├── sitemap.xml
├── send-email.php
├── css/
│   └── styles.css
├── js/
│   └── script.js
└── img/
    └── (все изображения)

ЧАСТЫЕ ПРОБЛЕМЫ И РЕШЕНИЯ:
---------------------------

1. КОНТЕНТ СЪЕЗЖАЕТ ВНИЗ, ШАПКА НЕ ВИДНА
   
   ПРИЧИНА: CSS файл не загружается или неправильный путь
   
   РЕШЕНИЕ:
   - Откройте сайт в браузере
   - Нажмите F12 (открыть инструменты разработчика)
   - Перейдите на вкладку "Console" (Консоль)
   - Посмотрите, есть ли ошибки красным цветом
   - Перейдите на вкладку "Network" (Сеть)
   - Обновите страницу (F5)
   - Найдите файл "styles.css"
   - Если он красный или показывает ошибку 404 → путь неправильный
   
   ПРОВЕРЬТЕ:
   - Файл css/styles.css существует в папке /www/css/
   - В index.html путь указан как: <link rel="stylesheet" href="css/styles.css">
   - НЕ должно быть: /css/styles.css или ./css/styles.css (только css/styles.css)

2. ФОРМА НЕ ОТПРАВЛЯЕТСЯ
   
   ПРИЧИНА: PHP скрипт не работает
   
   РЕШЕНИЕ:
   - Убедитесь, что файл send-email.php находится в корне /www/
   - Проверьте права доступа: файл должен иметь права 644
   - На reg.ru PHP обычно работает автоматически

3. ИЗОБРАЖЕНИЯ НЕ ВИДНЫ
   
   ПРИЧИНА: Неправильные пути к изображениям
   
   РЕШЕНИЕ:
   - Все изображения должны быть в папке /www/img/
   - В HTML пути указаны как: img/hero.jpg (не /img/hero.jpg)

4. ПРАВА ДОСТУПА НА ФАЙЛЫ
   
   Для папок: 755
   Для файлов: 644
   
   Как проверить в FileZilla или через FTP:
   - Правой кнопкой на файл → Свойства/Права
   - Установите: 644 для файлов, 755 для папок

5. ПРОВЕРКА ПУТЕЙ К ФАЙЛАМ
   
   Откройте index.html в текстовом редакторе и проверьте:
   
   ✅ ПРАВИЛЬНО:
   <link rel="stylesheet" href="css/styles.css">
   <script src="js/script.js"></script>
   <img src="img/hero.jpg">
   
   ❌ НЕПРАВИЛЬНО:
   <link rel="stylesheet" href="/css/styles.css">
   <link rel="stylesheet" href="./css/styles.css">
   <link rel="stylesheet" href="http://domain.ru/css/styles.css">

БЫСТРАЯ ПРОВЕРКА:
-----------------
1. Откройте сайт в браузере
2. Нажмите F12
3. Вкладка "Console" - должны быть БЕЗ красных ошибок
4. Вкладка "Network" - все файлы должны загружаться (зеленые)
5. Если styles.css показывает 404 - проверьте путь к файлу

КОНТАКТЫ ДЛЯ ПОДДЕРЖКИ:
-----------------------
Если проблема не решается, пришлите:
1. Скриншот консоли браузера (F12 → Console)
2. Скриншот вкладки Network (F12 → Network)
3. Структуру папок на хостинге
